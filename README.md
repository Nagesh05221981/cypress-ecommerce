# Cypress E-commerce Automation

## Overview
This project contains end-to-end automated tests for an e-commerce web application using **Cypress**.  
It is designed to validate critical user flows such as login, product browsing, cart functionality, and checkout.

---

## Features
- User login and authentication tests  
- Product search and browsing  
- Shopping cart and checkout flow  
- Cross-browser testing with Cypress  
- Screenshots and videos for test reporting

---

## Tech Stack
- **Cypress** – End-to-end testing framework  
- **JavaScript / Node.js** – Test scripts  
- **Mocha & Chai** – Test runner and assertions  
- **Git & GitHub** – Version control and repository hosting
